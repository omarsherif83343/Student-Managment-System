# Student Management System

## Overview

Student Management System is a console-based application developed in C using Linked Lists. The system allows users to manage student records efficiently through a menu-driven interface.

## Features

* Create a student list.
* Add new student records.
* Insert students at the beginning of the list.
* Insert students in the middle of the list.
* Search for a student by ID.
* Update student information.
* Sort student records by:

  * Name
  * Department
  * Age
* Display all student records.
* Delete a specific student record.
* Delete all student records.

## Technologies Used

* C Programming Language
* Dynamic Memory Allocation
* Singly Linked List Data Structure
* File Organization and Modular Programming

## Student Information

Each student record contains:

* Student ID
* Student Name
* Department
* Age

## Learning Objectives

This project was developed to practice:

* Data Structures (Linked Lists)
* Dynamic Memory Management
* CRUD Operations (Create, Read, Update, Delete)
* Searching and Sorting Algorithms
* Menu-Driven Console Applications

## How to Run

1. Compile the project using GCC:

```bash
gcc main.c -o student_management
```

2. Run the executable:

```bash
./student_management
```
